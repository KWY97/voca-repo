- 음훈
- 주요 부수
- 한자 조합 원리
- 암기용 설명

🎬 집중 모드

집중 모드를 활성화하면

- 카테고리 UI 숨김
- 단어 목록 숨김
- 통계 숨김
- 단어 크게 표시

영화관 모드처럼 단어 암기에만 집중할 수 있습니다.

ESC 키를 누르면 종료됩니다.

📝 4지선다 퀴즈

카테고리에 따라 자동으로 적절한 문제 유형이 선택됩니다.

일반 단어

- 한자 → 히라가나
- 히라가나 → 한자
- 한자 → 뜻
- 뜻 → 한자

부사 및 기타

- 표기 → 뜻
